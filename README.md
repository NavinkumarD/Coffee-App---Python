# India Coffee App

A simple GUI application for ordering Indian coffees, built with Python and Tkinter.

## Features

- Display a menu of popular Indian coffees with prices
- Add coffees to your order with quantity selection
- View your current order summary
- Checkout and reset the order

## Indian Coffees Included

1. South Indian Filter Coffee - $2.50
2. Mysore Coffee - $3.00
3. Chikmagalur Coffee - $3.50
4. Coorg Coffee - $4.00
5. Karnataka Coffee - $2.75

## How to Run

### Option 1: Run the Python Script
1. Ensure you have Python installed (version 3.x recommended).
2. Install Tkinter if not already installed (usually comes with Python).
3. Run the app: `python main.py`

### Option 2: Use the Batch File
- Double-click `run.bat` to run the app.

## Requirements

- Python 3.x
- Tkinter (included with Python)

## Usage

1. The app will open a window displaying the coffee menu.
2. Click "Add to Order" to select a coffee by number and enter quantity.
3. Click "View Order" to see your current order.
4. Click "Checkout" to finalize and reset the order.

## Notes

- This is a simple desktop application for demonstration purposes.
- No data is saved; orders reset on checkout or app restart.
